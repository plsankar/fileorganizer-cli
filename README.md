## Install

Install this package gloably from npm repository.

```shell
npm i @plsankar/fileorganizer -g
```

## Usage

Run the cli command in the folder you want to organize.

```shell
fileorganizer <options>
```

## Options

`-d, --date`

Use Date Separation.

`-l, --log`

Generate Log File.
